I’ve once again improved my little [Last.fm](http://last.fm) script for use with [RubyOSA](http://rubyosa.rubyforge.org/). It needed updating badly. There were quite a few things bugging me.

What’s new:

• Nice console messages telling you what’s happening.  
• Support for skipping over identical genre’s.  
• Ability to continue or abort tagging process. So if you don’t like that tag, type ‘n’.  
• Use -q to subdue tagging confirmations.  
• Full UTF-8 support, as well as proper URL escaping.

[Download it now!](http://633k.net/tag_genre_lastfm.rb) And have somewhat useful genre’s to choose from.

I will tackle cleaning it up later. Add me to your RSS reader if you want script updates.

Remember, you can get RubyOSA with the Last.fm Tagger [here](http://rubyosa.rubyforge.org/).
